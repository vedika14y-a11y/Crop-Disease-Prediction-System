# Crop-Disease-Prediction-System
Machine Learning project for crop disease prediction using symptoms and temperature


## Project Overview
The Crop Disease Prediction System is a machine learning-based project designed to identify crop diseases using input features such as crop symptoms and temperature. 

The main objective of this project is to assist in early detection of plant diseases, which can help farmers take timely action and reduce crop losses. This project demonstrates how machine learning can be applied to solve real-world problems in the agriculture domain.

---

## Problem Statement
Crop diseases can significantly affect agricultural productivity. Early detection is often difficult without expert knowledge. This project aims to provide a simple predictive system that can help identify diseases based on available data.

---

## Dataset Description
The dataset used in this project contains information related to crop conditions.

### Features:
- Crop Symptoms  
- Temperature  

### Target Variable:
- Disease (type of crop disease)

The dataset is used to train the model to learn patterns between environmental conditions and disease occurrence.
